# Projects Dashboard

```dataview

file.link AS "Project",
goal as "Goal",
due AS "Due",
complete AS "Complete"
FROM "3. Projects"
WHERE file.name != "3. Projects"
WHERE project_page_type = "project"
```

## üë• Meetings 
---
 ```dataview
TABLE WITHOUT ID
file.link AS "Meeting",
 date AS "Date",
 time AS "Time",
 project AS "Project"
 FROM "4. Areas/Meeting Notes"
 WHERE project != ""
 WHERE file.name != "Meeting Notes"
 SORT file.name DESC
 LIMIT 10
 ```
